**标题:**
开源了一个 Claude Code 密钥保护工具，自动脱敏 183 种 API key / 数据库密码 / 钱包私钥

**正文:**

用 Claude Code 写代码时，Claude 可以直接读取你的文件——包括 .env 里的 API key、数据库连接串里的密码、甚至钱包私钥。

我做了一个 hook 工具 **Claude Secret Shield**，在 Claude 读文件时自动把密钥替换成占位符，写回文件时自动还原。

**举个例子：**

Claude 读到的内容：
```
OPENAI_API_KEY={{OPENAI_PROJECT_KEY_a1b2c3d4}}
DATABASE_URL={{POSTGRES_URL_e5f6a7b8}}
```

真实文件不受影响，Claude 写代码时占位符自动还原。

**覆盖范围：**
- 183 种密钥模式：OpenAI、Anthropic、AWS、GitHub、Stripe、Slack、数据库连接串、Web3 钱包私钥等
- 48 种敏感文件直接拦截：.env、credentials.json、id_rsa、hardhat.config.js 等
- 用户 prompt 中粘贴密钥也会被拦截

**一行安装：**
```bash
curl -fsSL https://raw.githubusercontent.com/tokligence/claude-secret-shield/main/install.sh | sh
```
