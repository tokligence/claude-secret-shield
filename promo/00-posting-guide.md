## 发帖前准备

1. **录一个 GIF** — 用 asciinema 或屏幕录制，展示 Claude 读文件时密钥被替换的效果（30秒以内）
2. **确保 README 配图正常显示** — 在 GitHub 上检查一遍
3. **准备好回复** — HN 和 Reddit 帖子发出后 1 小时内需要积极回复评论

## 间隔建议

- 不要同一天在所有平台发，间隔 1-2 天
- 先发 HN（最有爆发力），观察效果
- 再发 Reddit（两个 sub 可以同一天）
